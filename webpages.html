What is HTML?

HTML stands forHyperTextMarkup Language.it's a markup language that web
developers use to structure and describe the content of a webpage(not a
progarmming language).

HTML consists of elelments that describe different typrs of content:paragraphs,
links,headings,images,video,etc.Web browsers understand HTML and render
HTML code as websites.

In HTML,each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element
you can learn more at MDN Web Docs.
